# knowledge/design/

Fiches distillées que les skills `design-*` chargent à la demande. Pas un cours — un mémo actionnable.

## Comment utiliser

Chaque skill indique les fiches à charger dans sa section *Knowledge base*. Ouvrir **uniquement** la fiche nécessaire au moment où c'est nécessaire.

Pour aller plus loin sur un sujet, chaque fiche pointe vers la source brute dans `knowledge/design/skills-main/` ou `knowledge/design/designer-skills-main/`.
